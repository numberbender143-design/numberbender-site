OBS setup — the free backup option

Numberbender Studio runs right in your browser and needs no install. But if you already use OBS Studio (free), or you want it to appear as a real virtual camera inside Zoom/Teams, here's the same Teach It Like You Stream It two-pane layout for OBS.

1 · Download a starter scene

Import this in OBS, then point the sources at your own camera and window.

⬇ Windows scene ⬇ Mac scene Get OBS Studio (free) →

2 · Import it into OBS

  1. Open OBS → menu Scene Collection ▸ Import.
  2. Choose the file you downloaded, then Scene Collection ▸ pick Numberbender Studio.
  3. You'll see two scenes: NB · Side-by-side and NB · Spotlight (PIP).

3 · Point the sources at your devices

Scene files can't know your specific camera or window, so set them once:

  1. Double-click the Camera source → choose your webcam (or your phone / Continuity Camera).
  2. Double-click Screen / iPad / Slides → choose the window to show (your slides, whiteboard, or — for iPad on Mac — the QuickTime window).
iPad & tablet tips are the same as in the web app: Mac uses QuickTime (cable), Windows uses an AirPlay-receiver app, and a Wacom/Bamboo pad draws into a whiteboard app you capture.

4 · Go live

  1. Virtual camera: click Start Virtual Camera, then in Zoom/Teams pick OBS Virtual Camera as your camera.
  2. Or share the window: open Window Projector (right-click the preview) and screen-share that — just like the web app.
Not sure which to use? The browser version at numberbender.com/studio is simpler and needs zero install — OBS is here only if you prefer it or want the virtual-camera route.
